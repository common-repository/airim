== Description ==

Airim is an FAQ widget which is being rapidly adopted by high-traffic websites to accelerate inbound sales.

Airim helps website visitors find relevant information effortlessly and
guides them to the next step.

Features:

* Suggest visitors top five questions they are most likely to ask, based on webpage and past behavior.

* When users try to leave your page, ask them why and see your bounce rates drop
* Track all unanswered questions in one dashboard and add them to your FAQ list in one click.

* Start by importing from your FAQs page (or knowledge base) and manage them with a user-friendly interface.
Integrate seamlessly with your favorite internal or external tools


== Installation ==


1. Create an account on [Airim](https://airim.co/users/sign_up)
1. Enter your domain name and create / upload FAQs
1. Install Airim widget in wordpress
4. Thats it !
